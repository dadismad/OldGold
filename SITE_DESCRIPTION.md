# OldGold - Professional Site Description

## Project Overview

**OldGold** is a premium digital marketplace platform specializing in verified financial accounts. The platform connects buyers seeking established accounts with verified sellers, providing a secure environment for transactions across three main categories:

1. **Online Banks** - Revolut, Wise, N26, Monzo, Payoneer, Neteller, Paysera, Skrill, PayPal
2. **Crypto Exchanges** - Binance, Bybit, OKX, Coinbase, Kraken, KuCoin, Huobi, Gate.io, Bitget  
3. **Money Transfer Services** - Western Union, MoneyGram, Remitly, WorldRemit, XE, CurrencyFair, OFX

---

## Technical Specifications

### Technology Stack
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Theme System:** next-themes (Dark/Light mode)
- **Internationalization:** Custom i18n (English/Russian)

### Design System
- **Golden Ratio (φ = 1.618)** applied throughout for visual harmony
- **Responsive Design:** Mobile-first approach with adaptive layouts
- **Dark Mode:** Full dark/light theme support
- **Accessibility:** ARIA labels, semantic HTML
